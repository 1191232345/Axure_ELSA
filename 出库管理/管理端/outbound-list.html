初始化页面（数据展示逻辑）

逻辑项 说明 数据来源 展示规则 备注
出库列表加载 页面加载时展示所有出库订单 出库订单表 按创建时间倒序排列，默认隐藏已出库订单 支持分页，每页10条
订单状态展示 根据订单状态显示不同的操作按钮 订单状态字段 待拣货/待揽收显示签出按钮，已出库显示已签出 -

检索条件

检索项 输入方式 逻辑说明
仓库代码 下拉选择框 根据选择仓库精确筛选
拣货类型 下拉选择框 精确匹配拣货类型
客户代码 下拉选择框 根据选择客户精确筛选
SKU 文本输入框+查询类型选择 精确查询：完全匹配；模糊查询：包含匹配
配送渠道 下拉选择框 精确匹配配送渠道
出库单号 文本输入框+查询类型选择 精确查询：完全匹配；模糊查询：包含匹配；大批量查询：批量输入多个单号，其他筛选条件自动禁用
参考号 文本输入框+查询类型选择 精确查询：完全匹配；模糊查询：包含匹配
跟踪号 文本输入框 包含匹配
创建时间 日期范围选择器 筛选创建时间在指定范围内的订单
隐藏已出库 下拉选择框 选择"是"时过滤掉已出库状态的订单

按钮逻辑

按钮 位置 触发动作 前置条件 后续操作
搜索 搜索区域右侧 根据筛选条件查询数据 无 显示筛选结果，重置分页到第1页
重置 搜索区域右侧 清空所有筛选条件 无 显示所有数据，恢复大批量查询模式前的筛选条件可用状态
展开/收起 搜索区域右侧 切换高级筛选条件的显示/隐藏 无 显示或隐藏创建时间、隐藏已出库筛选条件
批量签出 操作区域左侧 打开确认签出弹窗 至少选中一条订单 确认后批量更新订单状态为"已出库"，刷新列表
签出 每行操作列 打开确认签出弹窗 订单状态为待拣货或待揽收 确认后更新订单状态为"已出库"，刷新列表
复制按钮 单号、参考号、跟踪号后 点击复制按钮，将对应单号复制到剪贴板 无 显示Toast提示"已复制到剪贴板"
查看详情 每行操作列 跳转到订单详情页面 无 跳转到outbound.html页面，传递订单号参数

属性取值逻辑（主表）

字段 说明 数据来源 取值规则 显示格式 存储长度 属性存储类型
出库单号 出库订单唯一标识 系统生成 格式：DEMO-时间戳-随机字符串 文本显示 64 字符型
参考号 客户提供的订单参考号 键盘输入 客户自定义 文本显示 64 字符型

SKU及数量 订单包含的SKU和数量 订单明细表 格式：SKU编码 x数量，多个用逗号分隔 文本显示 256 字符型
拣货类型 拣货方式 手工选择 一单一件/一单多件 文本显示 32 枚举型

创建人 订单创建人 系统获取 当前登录用户 文本显示 32 字符型
订单状态 订单当前状态 系统更新 待拣货/待揽收/已出库/已取消/已送达 文本显示 32 枚举型
