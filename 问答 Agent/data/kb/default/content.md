# 海外仓计费规则说明

## 1.1 术语定义

- **头程**：货物从始发国至海外仓的跨境运输段，通常由商家或货代承担
- **尾程**：海外仓至最终收件人的本地配送段，按渠道价卡计费

## 3.3 超期库存计费

**超期库存**定义：库龄 **≥ 180 天**的在库商品。

- 自第 **181 天**起按 **阶梯费率**计费（详见附录 B）
- **90 天库龄预警**仅为提醒，不产生超期费用
- 计费起始日以 WMS **上架日期**为准

## 3.4 阶梯费率（摘要）

| 库龄 | 费率（USD/立方米/天） |
|------|----------------------|
| 181–270 天 | 0.15 |
| 271–365 天 | 0.25 |
| 366 天及以上 | 0.40 |

具体以合同及当期价卡为准。

# 禁运品与合规清单

## 2.2 德国仓禁运类别

德国仓 **禁运** 主要包括：

- 易燃易爆品、未申报液体
- 无合规文件的 **电池**（需 UN38.3）
- **仿牌**商品
- 无资质 **食品**

## 3.1 法国仓带电产品规则

法国仓 **带电产品** 入库须：

- **CE 认证**
- 电池类提供 **UN38.3** 报告
- 系统申报带电属性，否则拒收

## 4.0 通用合规

各国仓别禁运清单 **不可混用**；入库前须在系统中选择正确目的仓并核对最新合规公告。

# 欧洲仓退货处理 SOP

## 2.0 标准流程

FBA 退货入库：**创建退货单 → 仓库收货 → 质检 → 上架或销毁**，标准时效 **5–7 个工作日**。

## 3.2 德国仓质检标准

德国仓退货质检项目：

- 外箱完整性、SKU 与系统预报匹配
- 外观瑕疵分级：**A 级**可直接上架；**B 级**需商家确认；**不合格**隔离
- 不合格品 **48 小时内**通知商家，逾期按 SOP 默认销毁

## 3.4 超期异常处理

德国仓退货若 **超过 7 个工作日**仍未闭环：

1. 系统自动标记异常并通知商家与客服
2. 货物转入 **隔离区**
3. 需 **主管审批** 后续方式：退回 / 销毁 / 特批上架

## 4.1 各国时效对比

| 仓库 | 退货处理周期 |
|------|-------------|
| 英国仓 | 5–7 个工作日 |
| 德国仓 | 7–10 个工作日（含质检） |

流程均含：收货 → 质检 → 上架或销毁。不可将两国规则混为一谈。

# FBA 退货入库 SOP

## 2.0 标准流程

1. 商家在系统 **创建退货单** 并选择目的仓
2. 仓库 **收货登记**，核对 tracking 与 SKU
3. **质检分类**（可售 / 瑕疵 / 不可售）
4. **上架** 可售库存或 **销毁** 不可售品

标准整体时效：**5–7 个工作日**（自仓库签收日起算）。

## 2.1 异常件

无法识别 SKU 或面单破损的包裹，暂存 **待认领区**，**7 日**内无认领按无主货处理。

# 美国仓入库操作规范

## 3.2 截单时间表

标准件入库最晚截单：**工作日 14:00**（美国当地时区）。

- 截单前提交的系统入库单，当日安排卸货与登记
- 截单后提交的订单顺延至下一工作日处理
- 大件/非标件截单时间为 **12:00**，需提前在系统标注货物类型

## 5.1 外箱破损处理

入库时发现外箱破损，操作步骤：

1. **拍照留证**（外箱六面、破损点、面单）
2. 在 WMS 登记 **入库异常**，选择「外箱破损」
3. **通知商家**并在 2 小时内同步初步结论
4. 货物转入 **隔离区**，待商家确认是否拆检
5. **48 小时内**必须反馈处理意见，逾期按默认规则销毁或退回

## 5.2 数量差异

实收数量与预报不符时，按 ±2% 容差处理；超出容差须全量清点并升级客服。
