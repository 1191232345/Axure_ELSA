# AI客服管理系统 - 测试用例文档

## 1. 概述

### 1.1 测试范围
本测试用例覆盖AI客服管理系统的三个核心模块：
1. 客户代码与客户群管理
2. 客户群与账号关系管理
3. 群问答记录管理

### 1.2 测试环境
- 浏览器：Chrome 120+、Firefox 115+、Safari 17+
- 分辨率：1920x1080（桌面端）、375x667（移动端）
