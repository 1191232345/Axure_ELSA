# AI客服管理系统 - PRD文档

## 1. 需求概述

### 1.1 项目背景
本系统是一个逻辑AI客服项目，通过第三方平台中转微信消息，将消息传递给本地LLM（大语言模型），LLM返回的内容再通过第三方平台中转到微信群。

### 1.2 功能定位
- **客户代码与客户群管理**：建立客户代码与微信客户群的关联关系
- **客户群与账号关系管理**：管理客户群内的成员账号及其角色
- **群问答记录管理**：记录和管理客户群中的AI问答内容

### 1.3 核心流程
```
微信群消息 → 第三方平台 → 本地LLM → 第三方平台 → 微信群回复
```

## 2. 功能模块

### 2.1 客户代码与客户群管理

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 列表展示 | 展示客户代码、客户群名称、群ID、创建时间、状态 | P0 |
| 添加关联 | 添加客户代码与客户群的关联关系 | P0 |
| 编辑关联 | 修改已有的关联信息 | P1 |
| 删除关联 | 删除指定的关联记录 | P1 |
| 搜索筛选 | 按客户代码或群名称搜索 | P0 |

### 2.2 客户群与账号关系管理

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 列表展示 | 展示群名称、账号名称、账号ID、绑定时间、角色、状态 | P0 |
| 添加账号 | 向客户群添加成员账号 | P0 |
| 编辑账号 | 修改账号角色信息 | P1 |
| 移除账号 | 从群中移除成员账号 | P1 |
| 搜索筛选 | 按群名称或账号名称搜索 | P0 |

### 2.3 群问答记录管理

| 功能点 | 描述 | 优先级 |
|--------|------|--------|
| 列表展示 | 展示群名称、提问人、问题内容、AI回复、问答时间、评分 | P0 |
| 查看详情 | 查看完整的问答内容 | P1 |
| 重新回复 | 重新调用LLM生成回复 | P1 |
| 导出记录 | 导出问答记录数据 | P1 |
| 搜索筛选 | 按群名称、关键词、时间范围筛选 | P0 |

## 3. 数据结构

### 3.1 客户群关联数据

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | Number | 唯一标识 |
| customerCode | String | 客户代码 |
| groupName | String | 客户群名称 |
| groupId | String | 微信群ID |
| createTime | String | 创建时间 |
| status | String | 状态（normal/pending） |

### 3.2 账号关系数据

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | Number | 唯一标识 |
| groupName | String | 客户群名称 |
| accountName | String | 账号名称 |
| accountId | String | 账号ID |
| bindTime | String | 绑定时间 |
| role | String | 角色（群主/管理员/普通成员） |
| status | String | 状态（normal/exited） |

### 3.3 问答记录数据

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | Number | 唯一标识 |
| groupName | String | 客户群名称 |
| questioner | String | 提问人 |
| question | String | 问题内容 |
| answer | String | AI回复内容 |
| qaTime | String | 问答时间 |
| rating | Number | 评分（1-5星） |

## 4. 页面设计

### 4.1 页面结构
- **顶部导航栏**：系统名称、同步数据按钮
- **Tab切换区**：三个功能模块切换
- **主内容区**：表格展示、搜索筛选、分页

### 4.2 交互设计

#### Tab切换
- 点击Tab标签切换对应模块内容
- 当前活动Tab高亮显示

#### 搜索功能
- 实时搜索，输入即过滤
- 支持多条件组合搜索

#### 弹窗操作
- 添加/编辑操作通过模态框完成
- 操作成功后自动刷新列表

## 5. 技术实现

### 5.1 技术栈
- **前端框架**：原生HTML/CSS/JavaScript
- **样式框架**：Tailwind CSS 3.x
- **图标库**：Font Awesome 6.x
- **数据存储**：JSON文件 + 本地内存

### 5.2 API接口设计

#### 获取客户群列表
```
GET /api/data/customer-groups
```

#### 添加客户群关联
```
POST /api/data/customer-groups
{
    "customerCode": "string",
    "groupName": "string",
    "groupId": "string"
}
```

#### 获取账号关系列表
```
GET /api/data/account-relations
```

#### 获取问答记录列表
```
GET /api/data/qa-records
```

#### 重新生成AI回复
```
POST /api/data/qa-records/{id}/retry
```

## 8. 附录

### 8.1 状态定义
- **normal**：正常状态
- **pending**：待审核
- **exited**：已退出

### 8.2 角色定义
- **群主**：群管理员，最高权限
- **管理员**：辅助管理权限
- **普通成员**：普通群成员
