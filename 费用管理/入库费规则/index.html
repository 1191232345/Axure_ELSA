属性取值逻辑（新增弹窗）

字段 说明 数据来源 取值规则 显示格式 存储长度 属性存储类型
费用组 费用类型所属分类 系统生成 默认为当前选中的Tab分类，禁用不可修改 文本显示 64 字符型
费用组 * 费用类型所属的分类 手工选择 数据源:基础数据-物流产品管理-收费类型管理，取值=费用组 下拉选择 64 字符型
费用类型名称* 一级和二级费用类型的统一选择 手工选择（树形下拉框） 数据源:基础数据-物流产品管理-收费类型管理，取值=费用类型名称 树形下拉选择 64 字符型
计费单位 * 收费的计价单位 手工选择 从下拉列表选择：柜、票、件、箱、托、KG、个 下拉选择 32 字符型
失效时间 规则失效的时间 手工选择 可选，选择规则的失效时间，超过该时间后规则不再生效 日期选择 - DATE
费率标准 不同规格的价格标准 键盘输入 当费用类型为"卸货费"类（如整柜入库卸货费、快递散货入库卸货费等）时显示表格，支持添加多行：规格（如20GP）、价格（数字）、备注（文本） 表格录入 - JSON
阶梯价格 按数量范围的价格阶梯 键盘输入 当选择二级费用类型时显示表格，支持添加多行：开始量（数字）、结束量（数字）、价格（数字） 表格录入 - JSON
费率规则 费率规则文本描述 键盘输入 当费用类型为"附加费"类一级项（如整柜入库附加费）但未选择二级费用类型时显示，每行一条：规格|价格|备注 多行文本 512 字符型
重量阶梯 重量阶梯价格描述 键盘输入 可选，每行一条：范围|价格|单位 多行文本 512 字符型
条件说明 收费规则的适用条件 键盘输入 可选，描述收费规则的适用条件，如"单箱重量超过35kg时收取超重费" 多行文本 256 字符型
计费举例 计费示例说明 键盘输入 可选，提供计费示例说明，帮助用户理解计费规则 多行文本 512 字符型
备注 规则补充说明 键盘输入 可选，规则说明或特殊条件 多行文本 256 字符型

新增弹窗交互逻辑

触发条件 交互逻辑 展示内容
打开新增弹窗 1. 显示新增弹窗
2. 费用组为空，需要用户手动选择（入库、出库、仓储、其他）
3. 费用类型下拉框显示"请先选择费用组"		 显示新增弹窗，所有字段为空或默认值
选择费用组 1. 根据选择的费用组动态加载对应的费用类型树形数据
2. 费用类型下拉框显示该分类下的合并费用类型（如：整柜入库卸货费、整柜入库附加费等）
3. 一级项加粗显示，二级项缩进显示		 费用类型下拉框动态更新
选择一级费用类型（如：整柜入库卸货费） 1. 显示选择结果区域
2. 显示"费率标准"表格
3. 隐藏其他表单区域		 选择结果展示 + 费率标准表格
选择二级费用类型（如：SKU超量费、超重费） 1. 显示选择结果区域，包含一级和二级名称
2. 显示"阶梯价格"表格
3. 隐藏其他表单区域		 选择结果展示 + 阶梯价格表格
点击"添加费率"按钮 在费率标准表格中新增一行空白行 新增一行：规格输入框、价格输入框、备注输入框、删除按钮
点击"添加阶梯"按钮 在阶梯价格表格中新增一行空白行 新增一行：开始量输入框、结束量输入框、价格输入框、删除按钮
点击"保存"按钮 1. 校验必填字段
2. 收集表单数据
3. 自动设置创建人、创建时间
4. 保存到数据源
5. 刷新列表		 保存成功提示，关闭弹窗，刷新主表
