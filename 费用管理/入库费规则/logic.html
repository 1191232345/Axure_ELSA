触发条件交互逻辑展示内容
打开新增弹窗1. 显示新增弹窗 2. 费用组默认选中"入库" 3. 自动加载费用类型树并选中第一个选项 4. 根据费用类型自动判断计费维度并显示对应表单显示新增弹窗，费用组和费用类型有默认值，计费维度自动判断
选择费用组1. 根据选择的费用组动态加载对应的费用类型树形数据 2. 自动选中第一个费用类型 3. 自动判断计费维度并显示对应表单费用类型下拉框动态更新，计费维度自动判断
选择费用类型1. 判断费用类型级别和名称 2. 二级费用类型→计费维度="阶梯价格"，显示阶梯价格表格 3. 卸货费→计费维度="费率标准"，显示费率标准表格 4. 附加费→计费维度="阶梯价格"，显示阶梯价格表格 5. 其他→计费维度="固定价格"，显示固定价格表单计费维度自动判断，显示对应的费率录入表单
点击"添加费率"按钮在费率标准表格中添加一行新记录费率标准表格新增一行
点击"添加阶梯"按钮在阶梯价格表格中添加一行新记录阶梯价格表格新增一行
点击"保存"按钮1. 校验必填字段（费用组、费用类型、计费单位） 2. 根据计费维度校验对应表单数据 3. 收集表单数据 4. 保存到数据源 5. 刷新列表保存成功提示，关闭弹窗，刷新主表
费用组 收费项目 收费子项 计费维度 取值逻辑 计费单位 计费逻辑 备注
整柜入库 卸货费（入库） 柜型 取值 = 单票入库单的 '柜型' /柜 费用 = 柜型单价 × 1
附加费（入库） SKU超量费 SKU 取值 = 单票入库单，sku种类数 /个 费用 = 阶梯单价 × 超出SKU种类数
超重费 重量（总重量） 取值 = 单票入库单，各SKU总重量的和 /KG 费用 = 阶梯单价 × 超出重量 ELSA暂无指令下发功能
清点费 SKU 取值 = 单票入库单，相同箱号内，各sku的件数和 /件 费用 = 阶梯单价 × 件数
散货入库 卸货费（散货） 托盘（总托数） 取值 = 单票入库单各SKU '打托件数' 的和 /托 费用 = 阶梯单价 × 托数
入库附加费 拆托费 托盘（混托数） 取值 = 单票入库单【拆托信息】标记为"混托"的托盘数量 /托 费用 = 阶梯单价 × 混托托数 ELSA暂时无箱概念
分货费 箱数（混托箱数） 取值 = 单票入库单【拆托信息】标记为"混托"托盘具体箱数 /箱 费用 = 阶梯单价 × 混托箱数 ELSA暂时无箱概念
超重费 重量（总重量） 取值 = 单票入库单，各SKU总重量的和 /KG 费用 = 阶梯单价 × 超出重量
清点费 件（一箱多件） 取值 = 单票入库单，相同箱内号，各sku的件数和 /件 费用 = 阶梯单价 × 件数 ELSA暂不支持1箱多件
SKU超量费 SKU 取值 = 单票入库单，sku种类数 /个 费用 = 阶梯单价 × 超出SKU种类数
规格下拉选择
数据来源: 基础数据管理
约束条件: 必填
取值规则: 根据费用组动态加载
示例: 20GP、40GP、40HC...
价格数字输入
数据来源: 键盘输入
约束条件: ≥0，精度2位小数，必填
取值规则: 用户手动输入
示例: 350.00
开始量数字输入
数据来源: 键盘输入
约束条件: ≥0，整数，必填
取值规则: 阶梯价格起始值
示例: 0
结束量数字输入
数据来源: 键盘输入
约束条件: ≥开始量，整数，必填，可填∞
取值规则: 阶梯价格结束值
示例: 100 或 ∞
计费单位下拉选择
数据来源: 费用类型默认单位
约束条件: 必填
取值规则: 自动带出，可修改
可选: 柜、件、托、KG、SKU、张
备注文本输入
数据来源: 键盘输入
约束条件: 可选，≤256字符
取值规则: 规格说明、适用条件
示例: 标准柜型
条件说明多行文本
数据来源: 键盘输入
约束条件: 可选，≤256字符
取值规则: 触发条件、适用场景
示例: 单箱重量超过35kg时收取
卸货费（整柜）
柜型按柜型计费
费用 = 柜型单价 × 1
SKU超量费
SKU按SKU数量计费
费用 = 阶梯单价 × 超出SKU数
超重费
重量（总重量）按超出重量计费
费用 = 阶梯单价 × 超出重量
清点费
SKU按清点件数计费
费用 = 阶梯单价 × 件数
卸货费（散货）
托盘（总托数）按托盘数量计费
费用 = 阶梯单价 × 托数
分货费
箱数（混托箱数）按分货箱数计费
费用 = 阶梯单价 × 混托箱数
