属性取值逻辑（新增弹窗）

字段 说明 数据来源 取值规则 显示格式 存储长度 属性存储类型
费用组 费用类型所属分类 系统生成 默认为当前选中的Tab分类，禁用不可修改 文本显示 64 字符型
费用组 * 费用类型所属的分类 手工选择 数据源:基础数据-物流产品管理-收费类型管理，取值=费用组 下拉选择 64 字符型
费用类型（含二级）* 一级和二级费用类型的统一选择 手工选择（树形下拉框） 数据源:基础数据-物流产品管理-收费类型管理，取值=费用类型（含二级） 树形下拉选择 64 字符型
计费单位 * 收费的计价单位 手工选择 从下拉列表选择：柜、票、件、箱、托、KG、个 下拉选择 32 字符型
失效时间 规则失效的时间 手工选择 可选，选择规则的失效时间，超过该时间后规则不再生效 日期选择 - DATE
费率标准 不同规格的价格标准 键盘输入 当费用类型为"卸货费"类时显示表格，支持添加多行：规格、价格、备注 表格录入 - JSON
阶梯价格 按数量范围的价格阶梯 键盘输入 当选择二级费用类型时显示表格，支持添加多行：开始量、结束量、价格 表格录入 - JSON
费率规则 费率规则文本描述 键盘输入 当费用类型为"附加费"类一级项但未选择二级费用类型时显示 多行文本 512 字符型
重量阶梯 重量阶梯价格描述 键盘输入 可选，每行一条：范围|价格|单位 多行文本 512 字符型
条件说明 收费规则的适用条件 键盘输入 可选，描述收费规则的适用条件 多行文本 256 字符型
备注 计费示例说明 键盘输入 可选，提供计费示例说明 多行文本 512 字符型
备注 规则补充说明 键盘输入 可选，规则说明或特殊条件 多行文本 256 字符型
