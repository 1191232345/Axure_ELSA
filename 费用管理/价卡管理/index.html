初始化页面（数据展示逻辑）

逻辑项 说明 数据来源 展示规则 备注
价卡列表 页面加载时展示所有价卡数据 localStorage / packages数组 按创建时间倒序排列，最新创建的显示在最上方 首次加载时显示示例数据
包含费用项 统计价卡中各类型费用项的数量 packages.feeItems 按费用类型分组统计，格式：类型名(数量项) 多个类型用顿号分隔
状态显示 根据价卡状态显示不同标签 packages.status 草稿(灰色)、生效(绿色)、已作废(红色) 使用不同颜色区分状态
关联客户 显示价卡关联的客户数量 packages.customers 已关联显示蓝色标签（客户图标+数量），未关联显示灰色提示 点击操作列"关联客户"按钮可修改
关联仓库 显示价卡关联的仓库数量 packages.warehouses 已关联显示绿色标签（仓库图标+数量），未关联显示灰色提示 点击操作列"关联仓库"按钮可修改
空状态 无价卡数据时显示空状态提示 packages.length === 0 显示空状态图标和提示文案，提供"创建第一个价卡"按钮 引导用户创建第一条数据

检索条件

检索项 输入方式 逻辑说明
价卡名称 文本输入框 模糊匹配价卡名称，不区分大小写
状态 下拉选择框 精确匹配价卡状态，可选：全部/草稿/生效/已作废
客户代码 文本输入框 模糊匹配价卡关联的客户代码，筛选包含该客户代码的价卡
仓库代码 文本输入框 模糊匹配价卡关联的仓库代码，筛选包含该仓库代码的价卡
生效时间（起） 日期时间选择器 筛选生效时间大于等于该时间的价卡
生效时间（止） 日期时间选择器 筛选生效时间小于等于该时间的价卡

按钮逻辑

按钮 位置 触发动作 前置条件 后续操作
创建价卡 筛选区域下方 打开创建价卡弹窗 无 清空表单，初始化费用项表格，显示弹窗
查询 筛选区域 执行筛选查询 无 根据筛选条件过滤价卡列表并刷新表格
重置 筛选区域 清空筛选条件 无 清空所有筛选输入框，显示全部价卡
查看 操作列 打开价卡详情弹窗 价卡存在 展示价卡完整信息，包括基本信息、费用项明细、有效期
编辑 操作列 打开编辑价卡弹窗 价卡存在且状态不为"已作废" 填充现有数据到表单，允许修改并保存
作废 操作列 作废价卡 价卡存在且状态不为"已作废" 弹出确认对话框，确认后将状态改为"已作废"并刷新列表
关联客户 操作列 打开关联客户弹窗 价卡存在 使用穿梭框选择要关联的客户，保存后更新价卡的客户关联数据并刷新列表
关联仓库 操作列 打开关联仓库弹窗 价卡存在 使用穿梭框选择要关联的仓库，保存后更新价卡的仓库关联数据并刷新列表
保存 创建/编辑弹窗 保存价卡数据 价卡名称必填，至少选择一个费用项 验证数据完整性，保存到localStorage，关闭弹窗，刷新列表，提示成功消息
取消 创建/编辑弹窗 关闭弹窗 无 关闭弹窗，清空表单数据
新增行 费用项表格上方 添加费用项行 无 在费用项表格中新增一行空白记录
删除 费用项表格每行 删除该费用项行 无 从费用项表格中移除该行

关联操作逻辑

操作项 触发方式 弹窗内容 操作流程 保存逻辑
关联客户 点击操作列"关联客户"按钮 显示价卡名称、穿梭框（可选客户/已选客户）、搜索框、操作按钮 1. 从可选客户列表中选择客户

4. 可使用全选/清空按钮批量操作
5. 点击已选客户可移除 		点击保存按钮，更新价卡的customers字段，刷新主表显示关联数量
关联仓库 点击操作列"关联仓库"按钮 显示价卡名称、穿梭框（可选仓库/已选仓库）、搜索框、操作按钮 1. 从可选仓库列表中选择仓库
2. 点击右箭头按钮添加到已选列表
3. 可使用搜索框筛选仓库
4. 可使用全选/清空按钮批量操作
5. 点击已选仓库可移除 		点击保存按钮，更新价卡的warehouses字段，刷新主表显示关联数量

属性取值逻辑（主表）

字段 说明 数据来源 取值规则 显示格式 存储长度 属性存储类型
id 价卡唯一标识 系统生成 时间戳生成，确保唯一性 数字 13位 数值型
name 价卡名称 键盘输入 必填，不能为空，支持中英文、数字、特殊字符 文本 128 字符型
description 价卡描述 键盘输入 选填，支持多行文本 文本 512 字符型
feeItems 费用项列表 手工选择 至少包含一个费用项，每个费用项包含类型、名称、折扣方式等 JSON数组 - JSON
effectiveDate 生效时间 日期时间选择器 选填，格式：YYYY-MM-DD HH:mm:ss 日期时间 - DATETIME
expiryDate 失效时间 日期时间选择器 选填，格式：YYYY-MM-DD HH:mm:ss，应大于生效时间 日期时间 - DATETIME
status 状态 系统生成/手工操作 draft(草稿)、active(生效)、cancelled(已作废) 枚举值 - 枚举型
customers 关联客户 穿梭框选择 选填，通过操作列"关联客户"按钮设置，可关联多个客户 JSON数组 - JSON
warehouses 关联仓库 穿梭框选择 选填，通过操作列"关联仓库"按钮设置，可关联多个仓库 JSON数组 - JSON
createdAt 创建时间 系统生成 创建时自动生成，格式：YYYY-MM-DD HH:mm:ss 日期时间 - DATETIME

属性取值逻辑（创建/编辑价卡弹窗）

字段 说明 数据来源 取值规则 显示格式 存储长度 属性存储类型
价卡名称 价卡名称输入 键盘输入 必填，不能为空 文本输入框 128 字符型
价卡描述 价卡描述输入 键盘输入 选填，支持多行文本 多行文本框 512 字符型
收费类型 费用项分类 下拉选择 从入库费规则数据中加载，选择后联动收费项下拉 下拉选择框 - 字符型
收费项 具体费用项 下拉选择 根据收费类型动态加载对应的费用项列表 下拉选择框 - 字符型
折扣方式 费用项折扣设置 下拉选择+数值输入 无折扣/百分比折扣/指定扣减，选择后显示对应的数值输入框 下拉选择框+数值输入框 - 枚举型/数值型
生效时间 价卡生效时间 日期时间选择器 选填，默认当天00:00 日期时间选择器 - DATETIME
失效时间 价卡失效时间 日期时间选择器 选填，默认当天23:00 日期时间选择器 - DATETIME
立即生效 是否立即生效价卡 复选框勾选 勾选后状态设为"active"，否则为"draft" 复选框 - 布尔型
