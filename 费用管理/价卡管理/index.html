初始化页面（数据展示逻辑）

逻辑项 说明 数据来源 展示规则 备注
价卡列表 页面加载时展示所有价卡数据 localStorage / packages数组 按创建时间倒序排列，最新创建的显示在最上方 首次加载时显示示例数据
包含费用项 统计价卡中各类型费用项的数量 packages.feeItems 按费用类型分组统计，格式：类型名(数量项)。展示引用的入库费规则分类信息 多个类型用顿号分隔
入库费规则引用 展示价卡引用的入库费规则信息 packages.feeItems（关联入库费规则数据） 在价卡详情中展示每个费用项引用的入库费规则完整信息，包括：费用组分类、费用类型、单位等 通过费用类型字段关联入库费规则
创建人/更新人 显示价卡的创建人和最后更新人 packages.createdBy / packages.updatedBy 创建时记录创建人，更新时记录更新人 未更新时创建人与更新人相同
创建时间/更新时间 显示价卡的创建时间和最后更新时间 packages.createdAt / packages.updatedAt 格式：YYYY-MM-DD HH:mm:ss 未更新时创建时间与更新时间相同
空状态 无价卡数据时显示空状态提示 packages.length === 0 显示空状态图标和提示文案，提供"创建第一个价卡"按钮 引导用户创建第一条数据

检索条件

检索项 输入方式 逻辑说明
价卡名称 文本输入框 模糊匹配价卡名称，不区分大小写
创建人 下拉选择 精确匹配创建人，从已有价卡中提取创建人列表
更新人 下拉选择 精确匹配更新人，从已有价卡中提取更新人列表
创建时间 时间段选择 选择开始时间和结束时间，筛选该时间段内创建的价卡
更新时间 时间段选择 选择开始时间和结束时间，筛选该时间段内更新的价卡

按钮逻辑

按钮 位置 触发动作 前置条件 后续操作
创建价卡 筛选区域下方 打开创建价卡弹窗 无 清空表单，初始化费用项表格，显示弹窗
查询 筛选区域 执行筛选查询 无 根据筛选条件过滤价卡列表并刷新表格
重置 筛选区域 清空筛选条件 无 清空所有筛选输入框，显示全部价卡
查看 操作列 打开价卡详情弹窗 价卡存在 展示价卡完整信息，包括基本信息（创建人、创建时间、更新人、更新时间）、费用项明细（费用组、费用类型、单位、单价、折扣方式、折扣金额、预计金额、备注）。费用项明细中展示引用的入库费规则完整信息，包括：费用组分类、费用类型、单位等
编辑 操作列 打开编辑价卡弹窗 价卡存在 填充现有数据到表单（包括费用项的折扣方式、折扣金额、备注），允许修改并保存
保存 创建/编辑弹窗 保存价卡数据 价卡名称必填，至少选择一个费用项 验证数据完整性，创建时设置创建人/时间，更新时设置更新人/时间，保存到localStorage，关闭弹窗，刷新列表，提示成功消息
取消 创建/编辑弹窗 关闭弹窗 无 关闭弹窗，清空表单数据
新增行 费用项表格上方 添加费用项行 无 在费用项表格中新增一行空白记录，包含收费类型、费用类型、单位、折扣方式、折扣金额、备注字段
删除 费用项表格每行 删除该费用项行 无 从费用项表格中移除该行

字段 说明 数据来源 取值规则 显示格式 存储长度 属性存储类型

属性取值逻辑（创建/编辑价卡弹窗）

字段 说明 数据来源 取值规则 显示格式 存储长度 属性存储类型
价卡名称 价卡名称输入 键盘输入 必填，不能为空 文本输入框 128 字符型
价卡描述 价卡描述输入 键盘输入 选填，支持多行文本 多行文本框 512 字符型
费用组 费用项分类 下拉选择 从入库费规则数据中加载，选择后联动费用类型下拉。支持引用入库费规则中的费用组分类（如：整柜入库、快递散货入库、托盘入库） 下拉选择框 - 字符型
费用类型 具体费用项 下拉选择 根据费用组动态加载对应的费用项列表。引用入库费规则中的具体费用类型（如：卸货费、收货费、质检费等），选择后自动填充单位字段，并在价卡详情中展示引用的入库费规则完整信息 下拉选择框 - 字符型
入库费规则引用 引用的入库费规则信息 系统自动关联 当选择费用类型后，系统自动关联对应的入库费规则，在价卡详情中展示：费用组、费用类型、单位等完整规则信息 价卡详情弹窗 - 关联型
折扣方式 费用项折扣类型 下拉选择 无折扣/百分比/指定扣减/一口价 下拉选择框 - 枚举型
折扣金额 折扣具体数值 数值输入 根据折扣方式显示对应输入框，百分比：0-100，其他：大于0的数值 数值输入框 - 数值型
备注 费用项备注说明 键盘输入 选填，用于记录费用项的特殊说明 文本输入框 256 字符型
