初始化逻辑

逻辑项 说明 数据来源 展示规则 备注
价卡列表 页面加载时展示所有价卡数据 localStorage / packages数组 按创建时间倒序排列，最新创建的显示在最上方 首次加载时显示示例数据
包含费用组 统计价卡中各类型费用项的数量 packages.feeItems 按费用类型分组统计，格式：数量项。展示引用的入库费规则分类信息 多个类型用顿号分隔
入库费规则引用 展示价卡引用的入库费规则信息 packages.feeItems（关联入库费规则数据） 在价卡详情中展示每个费用项引用的入库费规则完整信息，包括：费用组分类、费用类型、单位等 通过费用类型字段关联入库费规则
统计信息展示 显示所有价卡的总数 前端计算 格式：共 X 个价卡 -
字段 说明 数据来源 取值规则 显示格式 存储长度 属性存储类型
价卡名称 价卡的名称标识 手工输入 新增/编辑时用户填写 文本显示 128 字符型
包含费用组 统计价卡中各类型费用项的数量 前端计算 按费用类型分组统计，格式：类型名(数量项) 文本显示 - 字符型
创建人 创建该价卡的用户 系统生成 新增时自动获取当前登录用户 文本显示 64 字符型
创建时间 价卡创建的时间 系统生成 新增时自动生成当前时间 YYYY-MM-DD HH:mm:ss - DATETIME
更新人 最后更新该价卡的用户 系统生成 编辑保存时自动获取当前登录用户 文本显示 64 字符型
更新时间 价卡最后更新的时间 系统生成 编辑保存时自动更新为当前时间 YYYY-MM-DD HH:mm:ss - DATETIME
按钮 位置 触发动作 前置条件 后续操作
创建价卡 筛选区域下方 打开创建价卡弹窗 无 清空表单，初始化费用项表格，显示弹窗
查询 筛选区域 执行筛选查询 无 根据筛选条件过滤价卡列表并刷新表格
重置 筛选区域 清空筛选条件 无 清空所有筛选输入框，显示全部价卡
查看 每行操作列 打开价卡详情弹窗 价卡存在 展示价卡完整信息，包括基本信息（创建人、创建时间、更新人、更新时间）、费用项明细（费用组、费用类型、单位、单价、折扣方式、折扣金额、预计金额、备注）。费用项明细中展示引用的入库费规则完整信息，包括：费用组分类、费用类型、单位等
编辑 每行操作列 打开编辑价卡弹窗，预填充当前数据 价卡存在 填充现有数据到表单（包括费用项的折扣方式、折扣金额、备注），允许修改并保存
删除 每行操作列 弹出确认对话框 价卡存在 确认后删除，刷新列表

检索项 输入方式 数据来源 取值规则 备注

更新时间 时间段选择 手工选择 选择开始时间和结束时间，筛选该时间段内更新的价卡 支持单边筛选
