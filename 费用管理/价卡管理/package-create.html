交互逻辑

触发条件 交互逻辑 展示内容
页面加载完成 1. 清空所有表单字段
2. 初始化费用项表格为空
3. 自动新增一行费用项
4. 默认选中"入库"费用组Tab 		表单为空，费用项表格有一行空数据，"入库"Tab激活
切换费用组Tab 1. 切换Tab激活状态
2. 根据选中的费用组，动态加载对应的费用类型数据
3. 费用类型下拉框更新为该费用组下的选项
4. 只显示当前费用组的费用项 		Tab样式切换，费用类型下拉框动态更新，费用项表格过滤显示
选择费用类型 1. 自动检测是否为阶梯收费类型
2. 如果是阶梯收费，显示"配置"按钮
3. 如果是标准收费，显示折扣方式下拉框 		根据费用类型自动显示相应的折扣配置界面
点击"配置"按钮（阶梯收费） 1. 打开阶梯折扣配置弹窗
2. 显示阶梯价格预览
3. 提供折扣模式选择（折扣率、增减、一口价）
4. 实时计算预计价格 		显示阶梯折扣配置弹窗，支持配置折扣并实时预览
保存阶梯折扣配置 1. 验证折扣配置有效性
2. 保存配置到费用项
3. 关闭配置弹窗
4. 更新费用项表格显示 		配置成功提示，弹窗关闭，费用项表格更新
点击"新增行"按钮 1. 在费用项表格中添加一行新记录
2. 新行默认选中当前激活的费用组
3. 新行的费用类型默认选中第一个选项 		费用项表格新增一行数据
点击"删除行"按钮 删除当前行的费用项记录 费用项表格删除当前行
点击"保存"按钮 1. 校验必填字段（价卡名称）
2. 校验费用项数据完整性
3. 收集表单数据
4. 生成价卡ID（时间戳）
5. 设置创建人、创建时间、更新人、更新时间
6. 跳转到列表页 		保存成功提示，跳转到价卡列表页
点击"取消"按钮 跳转回价卡列表页，不保存数据 返回列表页，不保存数据
逻辑项 说明 数据来源 展示规则 备注
弹窗打开 点击"配置"按钮打开阶梯折扣配置弹窗 前端逻辑 弹窗居中显示，背景遮罩 仅对阶梯收费类型显示
阶梯价格数据加载 从入库费规则数据中加载该费用类型的阶梯价格 入库费规则数据 显示原始阶梯价格列表 数据源：inboundFeeRuleData
默认折扣模式 弹窗打开时默认选中"折扣率"模式 前端逻辑 折扣率单选框默认选中，折扣单位显示"%" 默认折扣值：100%
价格预测计算 根据默认折扣配置计算每个阶梯的预计价格 前端计算 实时显示预计价格列 预计价格不允许为负数
字段 说明 数据来源 取值规则 显示格式 备注
引用阶梯数 显示该费用类型包含的阶梯数量 入库费规则数据 统计originalTierPricing数组长度 徽章显示："引用 N 个阶梯" 只读
折扣模式 选择折扣的计算方式 手工选择 单选：折扣率、增减、一口价 单选按钮组 必选，默认"折扣率"
整体折扣方式 输入折扣值 手工输入 数字输入框，支持小数点后两位 输入框 + 单位（% 或 $） 必填，默认10
规格 显示阶梯的规格范围 入库费规则数据 格式化显示阶梯范围 表格列：如"1-100件" 只读
原价格 显示阶梯的原始价格 入库费规则数据 格式化显示价格 表格列：如"$100.00" 只读
预计价格 根据折扣配置计算的最终价格 前端计算 根据折扣模式和折扣值实时计算 表格列：如"$90.00" 实时更新，不允许为负数
触发条件 交互逻辑 展示内容
切换折扣模式 1. 更新折扣单位显示（% 或 $）
2. 清空折扣值输入框
3. 重新计算预计价格 		折扣单位切换，预计价格实时更新
输入折扣值 1. 实时计算每个阶梯的预计价格
2. 如果预计价格为负数，自动设为0
3. 更新预计价格列显示 		预计价格列实时更新
点击"保存"按钮 1. 验证折扣配置有效性
2. 收集折扣模式和折扣值
3. 计算最终阶梯价格
4. 保存到费用项的tierDiscountConfig
5. 关闭弹窗
6. 更新费用项表格显示 		保存成功提示，弹窗关闭，费用项表格更新显示折扣配置
点击"取消"按钮 关闭弹窗，不保存配置 弹窗关闭，配置不保存
点击遮罩层 关闭弹窗，不保存配置 弹窗关闭，配置不保存
点击右上角"×"按钮 关闭弹窗，不保存配置 弹窗关闭，配置不保存

数据保存逻辑

数据项 数据来源 存储位置 存储格式 备注
价卡ID 系统生成 localStorage.packages 时间戳（Date.now()） 唯一标识
价卡名称 表单输入 localStorage.packages 字符串 必填
价卡描述 表单输入 localStorage.packages 字符串 可选
费用项列表 表单收集 localStorage.packages 数组对象 包含费用组、费用类型、折扣配置等
创建人 系统生成 localStorage.packages 字符串（默认"管理员"） 自动填充
创建时间 系统生成 localStorage.packages ISO 8601格式 自动填充
更新人 系统生成 localStorage.packages 字符串（默认"管理员"） 自动填充
更新时间 系统生成 localStorage.packages ISO 8601格式 自动填充
