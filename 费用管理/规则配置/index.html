页面逻辑说明

初始化页面（数据展示逻辑）

逻辑项 说明 数据来源 展示规则 备注
规则配置列表 页面加载时展示所有规则配置数据 localStorage / ruleConfigs数组 按创建时间倒序排列，最新创建的显示在最上方 首次加载时显示示例数据
引用价卡 显示配置引用的价卡名称 ruleConfigs.price_card_name 显示价卡模板名称 点击可查看价卡详情
折扣配置 显示费用项折扣统计 ruleConfigs.fee_discounts 按费用类型分组统计，格式：类型名(数量项) 多个类型用顿号分隔
微调状态 显示折扣是否被微调 ruleConfigs.is_adjusted 已微调显示橙色标签，未微调显示灰色标签 对比价卡原始折扣与当前折扣
生效周期 显示配置的有效期 ruleConfigs.effective_start_time / effective_end_time 格式：YYYY-MM-DD HH:mm 至 YYYY-MM-DD HH:mm 根据当前时间判断是否生效中
空状态 无规则配置数据时显示空状态提示 ruleConfigs.length === 0 显示空状态图标和提示文案，提供"创建第一个规则配置"按钮 引导用户创建第一条数据

检索条件

检索项 输入方式 逻辑说明
客户 下拉选择 精确匹配客户ID，筛选该客户的规则配置
仓库 下拉选择 精确匹配仓库ID，筛选该仓库的规则配置
业务类型 下拉选择 精确匹配业务类型，支持全部业务、入库、出库、转仓、退货
创建人 文本输入 模糊匹配创建人名称，不区分大小写

按钮逻辑

按钮 位置 触发动作 前置条件 后续操作
创建规则配置 筛选区域下方 打开创建规则配置弹窗 无 清空表单，初始化费用项表格，设置默认生效时间，显示弹窗
查询 筛选区域 执行筛选查询 无 根据筛选条件过滤规则配置列表并刷新表格
重置 筛选区域 清空筛选条件 无 清空所有筛选输入框，显示全部规则配置
查看 操作列 打开规则配置详情弹窗 规则配置存在 展示完整信息，包括基本信息、引用价卡、折扣配置、生效周期
编辑 操作列 打开编辑规则配置弹窗 规则配置存在 填充现有数据到表单，加载折扣配置，允许修改并保存
保存 创建/编辑弹窗 保存规则配置数据 配置名称、客户、仓库、价卡、生效周期、创建人必填；客户+仓库组合唯一 验证数据完整性，检查唯一性约束，保存到localStorage，关闭弹窗，刷新列表，提示成功消息
取消 创建/编辑弹窗 关闭弹窗 无 关闭弹窗，清空表单数据
删除 操作列 删除规则配置 规则配置存在 弹出确认对话框，确认后删除数据，刷新列表，提示成功消息
重置测试数据 筛选区域下方 重置所有数据 无 清除localStorage，重新加载测试数据，刷新页面

属性取值逻辑（主表）

字段 说明 数据来源 取值规则 显示格式 存储长度 属性存储类型
id 规则配置唯一标识 系统生成 时间戳生成，确保唯一性 数字 13位 数值型
name 配置名称 键盘输入 必填，不能为空，支持中英文、数字、特殊字符 文本 128 字符型
customer_id / customer_name 客户ID和名称 下拉选择 必填，从客户列表中选择，与仓库组合唯一 文本 64 / 128 字符型
warehouse_id / warehouse_name 仓库ID和名称 下拉选择 必填，从仓库列表中选择，与客户组合唯一 文本 64 / 128 字符型
price_card_id / price_card_name 引用价卡ID和名称 下拉选择 必填，从价卡列表中选择，选择后加载折扣配置 文本 64 / 128 字符型
business_type 业务类型 下拉选择 选填，支持全部业务、入库、出库、转仓、退货 枚举值 - 枚举型
fee_discounts 费用项折扣配置 价卡加载+手工调整 从价卡加载初始折扣，可按费用类型微调，包含收费项、单位、折扣类型、折扣值、备注 JSON对象 - JSON
is_adjusted 微调状态 系统计算 对比价卡原始折扣与当前折扣，有差异则为true 布尔值 - BOOLEAN
effective_start_time 生效开始时间 日期时间选择 必填，格式：YYYY-MM-DD HH:mm 日期时间 - DATETIME
effective_end_time 生效结束时间 日期时间选择 必填，格式：YYYY-MM-DD HH:mm，需大于生效开始时间 日期时间 - DATETIME
created_by 创建人 键盘输入 必填，记录创建配置的用户 文本 64 字符型
created_at 创建时间 系统生成 创建时自动生成，格式：YYYY-MM-DD HH:mm:ss 日期时间 - DATETIME

属性取值逻辑（创建/编辑规则配置弹窗）

字段 说明 数据来源 取值规则 显示格式 存储长度 属性存储类型
客户代码 选择客户 下拉选择 必填，从客户列表中选择，选择后自动生成配置名称 下拉选择框 - 字符型
仓库代码 选择仓库 下拉选择 必填，从仓库列表中选择，选择后自动生成配置名称并检查唯一性 下拉选择框 - 字符型
价卡模板 选择引用的价卡 下拉选择 必填，从价卡列表中选择，选择后自动加载折扣配置 下拉选择框 - 字符型
费用项折扣 费用项折扣配置 价卡加载+手工调整 按费用类型分组显示，可新增/删除/修改费用项，包含收费项、单位、折扣类型、折扣值、备注 表格+输入框 - JSON
收费项 具体费用项 下拉选择 根据费用类型动态加载对应的费用项列表 下拉选择框 - 字符型

折扣值 折扣具体数值 数值输入 根据折扣类型显示对应输入框，百分比：0-100，其他：大于0的数值 数值输入框 - 数值型
备注 费用项备注说明 键盘输入 选填，用于记录费用项的特殊说明 文本输入框 256 字符型
名称 配置名称 键盘输入 必填，选择客户和仓库后自动生成建议名称 文本输入框 128 字符型
生效时间 配置生效开始时间 日期时间选择 必填，默认为当前时间 日期时间选择器 - DATETIME
失效时间 配置生效结束时间 日期时间选择 必填，需大于生效开始时间 日期时间选择器 - DATETIME
