页面逻辑说明

初始化页面（数据展示逻辑）

逻辑项 说明 数据来源 展示规则
列表数据 页面加载时展示全部记录 data/module-data.json / localStorage 按 createdAt 倒序，分页每页 10 条
空状态 无数据时显示空状态 records.length === 0 显示图标 +「暂无数据」+ 新增引导按钮

检索条件

检索项 输入方式 逻辑说明
状态 下拉选择 精确匹配 status 字段，空值表示全部
关键字 文本输入 模糊匹配 name 字段，不区分大小写

模块按钮逻辑

按钮 位置 前置条件 后续操作
搜索 筛选区 无 按条件过滤列表，重置到第 1 页
重置 筛选区 无 清空筛选条件，恢复全量数据
新增 列表上方 无 打开表单弹窗，保存后刷新列表
编辑 行操作列 记录存在 预填表单，保存后更新记录
删除 行操作列 用户确认 移除记录，刷新列表，Toast 提示

属性取值逻辑

字段 数据来源 取值规则 显示格式
名称 record.name 直接取值 文本，超长截断显示省略号
状态 record.status active→启用，inactive→禁用 绿色/灰色标签
创建时间 record.createdAt ISO 8601 转本地时间 YYYY-MM-DD HH:mm
弹窗属性描述
字段 输入方式 必填 取值规则
名称 文本输入 是 1-50 字符，去除首尾空格
状态 下拉选择 是 active（启用）/ inactive（禁用）
